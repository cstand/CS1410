Exercise: missingLetters

Description

John is playing a dice game. The rules are as follows.

  1. Roll two dice.
  2. Add the numbers on the dice together. (Do not use the sum() function)
  3. Add the total to your overall score.
  4. Repeat this for three rounds.

Function Name

dice_game

Parameters

  • rolls : a list of lists

Return Value

An integer representing the overall score.

Examples

dice_game([[1, 2], [3, 4], [5, 6]]) ➞ 21

dice_game([[1, 1], [5, 6], [6, 4]]) ➞ 0

dice_game([[4, 5], [4, 5], [4, 5]]) ➞ 27
